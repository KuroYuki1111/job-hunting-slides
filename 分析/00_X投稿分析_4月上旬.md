| **権威づけ** | 「内定N社の先輩が〜」 | 「大手内定者にまとめてもらった」 |
| **更新頻度** | 不定期 | **ほぼ毎日3投稿** |
| **内容の深さ** | テンプレ提供中心 | 考え方・ロジックまで解説 |

### aya_jinnjiの投稿パターン

**タイトルの定型文：**
```
① 面接官「質問文」 + すぐ活用して面接で使える内定者の回答シリーズ↓
② 【テーマ名】 + 悩みの引用 + 解決策の提示
③ 大手内定者に聞いた「〜」 + はこれ↓ / をまとめました↓
```

**本文の構造：**
```
【問題提起】 悩みのDMが来た / 相談をもらった
【権威づけ】 大手内定者にまとめてもらった / 内定者にヒアリングした
【ポイント】 箇条書き3〜5項目
【具体例】  面接官Q→内定者Aの対話形式
【締め】    保存しておいてね / 参考にしてね / SoKuMeeで練習しよう
```

**頻出フレーズ：**
- 「大手内定者に〇〇をまとめてもらいました」
- 「〜と相談/悩みが来たので」（読者の声起点）
- 「面接は『準備量』で差をつけろ」
- 「ぜひ参考にしてね」「保存しておいてね」
- 「SoKuMeeで面接練習しよう」
