# TikTok就活コンテンツ 文体・表現分析

> tiktok_ocr.db 83件（3アカウント）の分析結果

---

## 1. 全体に共通する「バズる文体」の特徴

### ① タイトル（冒頭1〜2行）= 数字 × 権威 × 緊急性

ほぼ全投稿が以下のパターンを組み合わせている：

| パターン | 実例 |
|---------|------|
| **内定N社の実績** | 「大手8社内定の先輩が使っていた」「10社内定もらった25卒が〜」 |
| **年次の先輩→後輩構図** | 「25卒→26卒へ」「大手内定した25卒内定者たちから」 |
| **知らないとヤバい煽り** | 「これ知らないで就活は相当やばい」「マジでもっと早く知りたかった」 |
| **テンプレ・模範解答の約束** | 「真似するだけで受かった」「丸パクリOK」「最強テンプレ」 |
| **質問文そのまま提示** | 「『なぜその大学？』面接官が納得する模範解答」 |

**→ 「誰が（権威）」「何を（具体質問）」「どうなる（結果）」を1〜2行で完結させている。**

---

### ② 文の長さとテンポ

**1文が短い。体言止め・倒置法が多用される。**

```
これガチで凄いから
試してみて
```
```
気持ちよくさせて、
最後に素直さや感謝の言葉で
絶頂させてfin
```
```
知らないとヤバい
```

- 1スライド = 1メッセージ = 1ポイントの原則
- 長くても3〜4行で改行が入る
- 説明文の合間に**1行のツッコミ・感想を挟む**ことでリズムを作る

---

### ③ 話し言葉ベース（書き言葉を避ける）

| 書き言葉（使わない） | 話し言葉（実際の表現） |
|---------------------|---------------------|
| 〜することが重要です | 〜が大事。マジで。 |
| 対策を行うべきです | 対策しないと勿体無い |
| 推奨します | 超オススメだよ |
| 効果的でした | ヤバかった |
| 〜と考えられます | 〜って思った… |

**特徴的な語尾パターン：**
- 「〜ダヨ」「〜だヨ」「〜ヨ」（カタカナ語尾）
- 「〜ですッ」「〜ございますッ」（促音）
- 「〜してた」「〜だった」（過去形の口語）
- 「〜してみて」「〜してね」（命令ではなくお願い調）

---

### ④ 感情のジェットコースター構造

1枚目（スライド）から最後まで、感情を揺さぶる構造が共通：

```
【Hook】  衝撃の事実・数字で引く
【Pain】  「あなたもこうなってない？」で不安を煽る
【How】   具体的なテンプレ・手順を見せる
【Proof】 「これで内定N社取れた」と結果で裏付ける
【CTA】   「やってみて」「検索してみて」で行動を促す
```

---

### ⑤ 「裏話感」「内輪感」の演出

読者を「特別な情報を知った側」に立たせる表現が多い：

- 「ここだけの話だけど」
- 「キャリセンでは教えてくれなかった」
- 「元面接官がこっそり教えてくれた」
- 「バ先の就活強かった先輩がこっそり教えてくれた」
- 「大手人事をしてる友達が〜」

**→ 「公式には出回らない裏情報」というフレーミング。**

## 3. 表現テクニック一覧

### よく使われる「パワーワード」

| カテゴリ | ワード |
|---------|--------|
| 結果を示す | 無双、内定N社、通過率90%、爆上がり、グーンと |
| 緊急性 | マジで、ガチで、絶対、今すぐ、知らないとヤバい |
| 秘匿性 | こっそり、ここだけの話、秘伝の、裏技、本当は教えたくない |
| 簡単さ | 真似するだけ、丸パクリOK、テンプレ、コピペ、これだけ |
| 権威 | 元面接官、大手人事、内定者、先輩、友達 |

### 構文パターン

```
① 【権威】が【していた/教えてくれた】【具体的なもの】
   例：「大手8社内定の先輩が使っていた逆質問」

② 【状況・質問文】＋【結果ワード】
   例：「『なぜその大学？』面接官が納得する模範解答」

③ 【否定】→【正解提示】
   例：「資格がない人は資格を取るべき 🙅‍♂️」→正しい考えを解説

④ 【数字】×【限定】
   例：「この10問だけは絶対に対策して」
```

## 4. まとめ：ヒットする就活TikTokの法則

1. **タイトルで9割決まる** — 数字・権威・質問文の3点セット
2. **1スライド1メッセージ** — 情報密度を下げ、スワイプのリズムを作る
3. **話し言葉で書く** — 「です・ます」より「〜だよ」「〜してみて」
4. **テンプレは穴埋め式** — 読者が即コピペできる形にする
5. **裏話フレーミング** — 「キャリセンでは教えてくれない」で希少性を出す
6. **失敗→成功のストーリー** — 共感→安心→行動の流れ
7. **感情ワードをスパイスに** — 「ヤバい」「無双」「爆上がり」を要所に配置
